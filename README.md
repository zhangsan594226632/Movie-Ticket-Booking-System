# Movie-Ticket-Booking-System
【万字文档+源码】基于SpringBoot+Vue电影售票系统-可用于毕设-课程设计-练手学习-学习资料分享-本文介绍了一个基于SpringBoot+Vue3的前后端分离电影票购买系统设计方案。系统分为用户前台和管理后台两大模块，采用JWT进行权限控制。前台功能包括：用户注册登录、影片浏览筛选、购物车管理、在线购票、影评交流和订单管理；后台功能涵盖：用户管理、影片分类管理、放映厅配置、电影信息维护、评论审核及订单处理等。系统采用MySQL数据库存储核心数据，包含用户表、电影分类表、放映厅表、电影信息表、购物车表和订单表等主要数据表。前端使用Vue3+ElementPlus构建，后端基于SpringBoot框架开发，
